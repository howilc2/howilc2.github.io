How to Install Titanium Build on Kodi

Before you get started, ensure that Kodi is installed on your device. Then, follow these steps:

  1. Open Kodi and go to the System menu
  2. Select File Manager and click on ‘Add Source’ on the left pane.
  3. In the pop-up window, click on ‘None’ and enter this URL: http://repo.supremebuilds.com
  4. Name the source as ‘Supreme Builds’and click ‘OK’.
  5. Return to the Kodi home screen and open the Add-ons menu.
  6. Select the box icon, followed by ‘Install from zip file’.
  7. Select the ‘Supreme Builds’ source and then click on the ‘repository.supremebuilds-x.x.x.zip’ file to install the Supreme Builds repository.
  8. Return to the Add-ons menu and click on ‘Install from repository’.
  9. Select the Supreme Builds Repository from the list.
  10. Click on ‘Program add-ons’ and then select the ‘Supreme Builds Wizard’ option.
  11. Select the version of Titanium Build you want to install.
  12. Wait for it to download and install the build.
  13. Restart Kodi, and you’re all set to enjoy the Titanium Build.
