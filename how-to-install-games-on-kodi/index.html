How to Install Games on Kodi

The first step to install games on Kodi is to add a source to your file manager in Kodi. To do this, follow these simple steps:

  1. Launch Kodi and click on the gear icon at the top of the home screen.
  2. Select ‘File manager’ from the list of options.
  3. Click on ‘Add source’ and then select ‘None’.
  4. Enter the following URL into the space provided: ‘http://fusion.tvaddons.co’ and click ‘Ok’.
  5. Give the source a name, such as ‘Fusion’, and click ‘Ok’.

The next step is to install the emulator and its add-on for running retro games. There are several emulators available for Kodi, but we recommend Retroarch as it supports a wide range of retro gaming consoles. To install Retroarch, follow these steps:

  1. Select ‘Add-ons’ from the main screen menu and then click on the ‘Package Installer’ icon in the top-left corner.
  2. Select ‘Install from repository’ and then click on ‘Kodi Add-on repository’.
  3. Click on ‘Game add-ons’ and then select ‘Retroarch’.
  4. Click on ‘Install’, and Retroarch will start downloading and installing.
  5. Once installed, click on ‘Retroarch’ and select the ‘Configure’ option.
  6. Select ‘Storage’ and then choose the directory where you have stored your retro games on your device.
  7. Finally, click ‘Save’ to finish the process.
